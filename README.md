# 뉴스 api 불러와서 카테고리별로 출력하기  


## 설치  

<b>1. 프로젝트 클론</b>  
```
$ git clone "https://github.com/ksmfou98/news-viewer.git"
```  

<b>2. 프로젝트 설치</b>
```
$ npm install
```    

<b>3. 프로젝트 실행</b>
```
$ npm start
```
